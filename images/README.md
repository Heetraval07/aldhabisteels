## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 
  - Hero images: 1920x1080px minimum (16:9 aspect ratio)
  - Service/Project images: 800x600px minimum (4:3 aspect ratio)
  - About/Fabrication images: 1200x800px minimum
- **Optimization**: Compress images for web (use tools like TinyPNG, ImageOptim)
- **Quality**: High quality, sharp, professional images from original site

## How to Extract Images

1. Visit https://aldhabisteel.com
2. Right-click on images and "Save Image As..."
3. Save with appropriate filenames listed above
4. If multiple images are available for a section, choose the best/most representative one
5. Crop/resize as needed while maintaining quality

## Notes

- DO NOT use stock photos, AI-generated images, or external visuals
- DO NOT add placeholder images - the site should only go live with real images from the existing site
- If a specific image is not available on the site, reuse an existing relevant image with appropriate cropping/framing
